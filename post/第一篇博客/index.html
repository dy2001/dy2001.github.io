  很久之前就想写博客了，但是一直再拖着。趁着今天比较闲，就用Hexo搭建了这个博客站点。

为什么写博客

  • 学过或者做过的东西长时间不用可能会忘记，所以说写博客当作写笔记是个不错的选择。
  • 平时也没有写过文章一类的，写这篇博客的时候还构思了很久，也可以提升一下自己的文笔。

博客搭建

  现在开源的博客系统非常多，看了看最后选择了 Hexo + Github Page,Hexo是一个基于Node.js的静态博客程序，一条命令就可以生成静态文件，push到github上就行了,至于Github Page嘛…可以省一笔服务器的钱🙄,访问速度也可以接受,然后还可以练练Markdown。

内容

  以技术分享为主，记录学习笔记和日常开发时遇到的问题和要点。
