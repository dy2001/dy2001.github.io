  逛别人博客的时候发现很多人的博客都有一个超级可爱的Live2D模型,赶紧给自己的博客也弄一个。
  好可爱！！！

第一步

  进入Hexo的根目录安装live2d插件，打开命令行输入：
npm install --save hexo-helper-live2d

第二步

  打开模型展示网站挑选你喜欢的模型，然后记住它的名字,也可以在这个页面查看所有的模型。

第三步

  执行命令npm install live2d-widget-model-+你喜欢的模型名，我用的模型叫 hijiki ,执行:
npm install live2d-widget-model-hijiki
